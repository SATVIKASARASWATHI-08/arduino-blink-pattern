# Blink LED Pattern using Arduino (Simulation)

## Description
This project demonstrates LED blinking using Arduino in Tinkercad.
The LED blinks three times and then pauses for 2 seconds.

## Components Used
- Arduino Uno
- LED
- Resistor (220 ohm)

## Concepts Used
- Digital Output
- for loop (repetition)
- delay (timing control)

## Tools Used
- Tinkercad Circuits

## Working
The LED turns ON and OFF repeatedly for 3 cycles using a loop.
After 3 blinks, the system pauses for 2 seconds before repeating.

## Learning Outcome
- Learned how to control an LED using Arduino
- Understood how loops work in embedded programming
- Practiced circuit simulation
